# Desarrollo Local

Inicia el servidor de desarrollo del proyecto BitcoinBaby.

## Comandos

```bash
# Verificar dependencias
pnpm install

# Iniciar desarrollo
pnpm dev
```

Si hay errores de dependencias, ejecuta `pnpm install` primero.
